About the Project

Project Description

This Flask-based Python application is designed to predict future stock prices using historical data and technical indicators. By integrating multiple functionalities such as user authentication, email verification, database management, and machine learning, it provides a comprehensive tool for stock analysis and prediction.

How is it Useful?

  • Stock Market Insights: Investors and traders can use this application to gain insights into potential future stock prices, helping them make informed investment decisions.
  • Technical Analysis: By incorporating various technical indicators, the application provides a detailed analysis of stock performance, which is crucial for technical traders.

Features of the Application

  • User Authentication and Verification: Ensures secure access through email-based verification and login.
  • Stock Data Retrieval: Fetches historical stock data from Alpha Vantage API, providing a solid foundation for analysis.
  • Technical Indicators: Calculates various technical indicators like Moving Averages (MA), Relative Strength Index (RSI), and Moving Average Convergence Divergence (MACD).
  • Machine Learning Model: Utilizes a RandomForestRegressor to predict future stock prices based on historical data and technical indicators.
  • Database Logging: Logs predictions and user data into a PostgreSQL database, ensuring persistent storage and retrieval.
  • Visualization: Generates interactive plots of predicted and actual stock prices using Plotly, enhancing data interpretation.
  • Holiday Check: Ensures predictions are not made for holidays, adding a layer of practicality to the model.

Detailed Code Description

The application consists of multiple modules and functionalities, each designed for specific purposes:

  • Imports and Configuration: Sets up the Flask application, API keys, and essential configurations.
  • Helper Functions: Includes functions for database connection, email verification, technical indicator calculation, and data retrieval.
  • Routes: Handles user login, email verification, and stock price prediction processes.

Usefulness of the Outputs

  • Predicted Prices: Provides future stock price predictions, helping users anticipate market movements.
  • Actual vs Predicted Prices Plot: Visual comparison between actual and predicted prices to gauge model accuracy.
  • Accuracy Score: R2 score indicating the model's predictive accuracy, helping users assess reliability.

Important Points for Users

  • Data Quality: Predictions depend heavily on the quality of historical data.
  • Technical Indicators: Understanding the indicators (MA, RSI, MACD) enhances interpretation.
  • Holiday Adjustments: Predictions account for holidays, ensuring practical outputs.
  • Security: Implements secure methods for user authentication and data protection.
